──────────────────────────────────────────
 NEW / EXTRA FEATURES I ADDED:
──────────────────────────────────────────

1. ZONE SCORING SYSTEM
   - Grid is divided into 3 strategic zones
   - Center zone  → 2× point multiplier (orange glow)
   - Edge zone    → 1.5× point multiplier (purple glow)
   - Corner zone  → 1.2× point multiplier (cyan glow)
   - Encourages strategic claiming, not just random clicks

2. POWERUP SYSTEM (full end-to-end)
   - Powerups randomly spawn on grid cells every 15 seconds
   - Claiming a cell with a powerup collects it automatically
   - 4 unique powerup types:
       ⚡ Speed Boost  — removes cooldown for 5 seconds
       💥 Mega Claim   — claims a 3×3 area at once
       🛡️ Shield        — blocks can't be stolen for 8 seconds
       👁️ Vision        — reveals upcoming powerup locations for 10s
   - Powerup cells pulse with a glowing animation on the grid
   - "My Powerups" panel shows collected powerups + timers
   - "On Grid" panel shows active powerup locations

3. TEAM MODE (Red vs Blue)
   - Players auto-assigned to Red or Blue team (balanced)
   - Team scores tracked separately in sidebar
   - Team badge shown on player card
   - Grid cells tinted by team color
   - Switch between FFA / Teams / Zones via toolbar

4. ACHIEVEMENT SYSTEM (10 badges)
   - 🚀 Pioneer       — claim your first block
   - 🏠 Settler       — own 10 blocks at once
   - ⚔️ Conqueror     — own 50 blocks at once
   - 👑 Emperor       — own 100 blocks at once
   - 🔥 On Fire       — 5-claim streak
   - ⚡ Unstoppable   — 10-claim streak
   - 🌟 Legendary     — 20-claim streak
   - 💯 Centurion     — reach 100 points
   - 🏆 Master        — reach 500 points
   - ⚡ Power Player  — use a powerup
   - Badges shown in right sidebar with tooltips
   - Toast notification fires when unlocked

5. CLAIM STREAK COUNTER
   - Consecutive claims tracked in real time
   - Streak counter shown on player card
   - Streak banner drops from top of screen at 3+
   - Max streak tracked server-side per player

6. MINIMAP
   - Live pixel canvas in bottom-left sidebar
   - Shows full 32×32 grid at a glance
   - Claimed cells colored by owner
   - Center zone overlay shown
   - Powerup positions shown as cyan dots
   - Click anywhere on minimap to jump the viewport there

7. LIVE IN-GAME CHAT
   - Real-time chat panel in right sidebar
   - Messages broadcast to all players via Socket.io
   - System messages for join / leave events
   - Input: press Enter to send, Escape to blur
   - Auto-scrolls to latest message
   - XSS-safe (HTML escaped)

8. FLOATING SCORE POPUPS
   - "+1", "+2" numbers float up from the cell you just claimed
   - Yellow for high-value zone cells, cyan for standard
   - CSS keyframe animation, auto-removed from DOM

9. TOAST NOTIFICATION SYSTEM
   - Slide-in toasts from top-right corner
   - 4 types with color-coded borders:
       Achievement  → gold border
       Powerup      → green border
       System       → default border
       Blocked      → red border
   - Auto-dismiss after 3.5 seconds

10. ZOOM + PAN CONTROLS
    - Zoom in/out buttons in grid toolbar (+ / −)
    - Zoom level display (40% → 300%)
    - Reset zoom button (⊙)
    - Keyboard shortcuts: + / - / 0
    - Grid viewport is scrollable / pannable

11. KEYBOARD SHORTCUTS
    - S      → toggle sound on/off
    - +/=    → zoom in
    - -      → zoom out
    - 0      → reset zoom
    - Enter  → focus chat input
    - Escape → blur chat input

12. WEB AUDIO SOUND EFFECTS
    - Synthesized tones via Web Audio API (no audio files needed)
    - Different tones for: claim, achievement, powerup collect,
      powerup activate, blocked by shield
    - Sound toggle button in header (🔊 / 🔇)

13. SHIELD CONFLICT RESOLUTION
    - If a player has an active Shield powerup, their blocks
      cannot be stolen by other players
    - Server validates this and sends "claim_blocked" event
    - Client shows a red toast: "Blocked! That player has a shield"

14. GRID RESET (Admin)
    - Reset button in header resets entire grid
    - Confirmation dialog before wiping
    - All scores / blocks / streaks reset to 0
    - All clients notified instantly

15. SERVER STATUS API
    - GET /api/status → returns live server stats (JSON)
    - GET /api/leaderboard → returns top scores (JSON)
    - Useful for monitoring or external integrations

16. STARFIELD BACKGROUND ANIMATION
    - 150 animated stars twinkling in the background
    - Randomized size, speed, opacity, delay
    - Nebula gradient overlays (purple + cyan + orange)
    - Pure CSS, zero performance cost

17. ENHANCED PLAYER CARD
    - Color-coded avatar (matches your claim color)
    - Glowing box-shadow matching player color
    - Shows: Score, Blocks owned, Current streak, Total claims
    - Team badge (Red/Blue) shown when in team mode

18. CONNECTION STATUS BADGE
    - "Live" with pulsing green dot when connected
    - "Offline" with red dot when disconnected
    - Auto-reconnect handled by Socket.io

19. CELL HOVER TOOLTIP
    - Hover any cell to see: owner name, zone type, point value
    - Shows powerup name if a powerup is on that cell
    - Shows cell ID number
    - Smart positioning (flips near screen edges)

20. RESPONSIVE + MOBILE LAYOUT
    - Sidebars collapse on screens < 860px
    - Grid cells shrink on mobile
    - Header pills hide on very small screens
    - Full touch support on mobile browsers


──────────────────────────────────────────
 TECHNICAL EXTRAS (Backend):
──────────────────────────────────────────

 - Server designed for 100+ concurrent players
 - Memory-efficient: ~5KB per player, ~50 bytes per cell
 - Claim cooldown: 300ms per cell (server-enforced)
 - Previous owner stat deduction when block is stolen
 - Auto-cleanup of user state on disconnect
 - Powerup spawner runs every 15s (max 5 on grid at once)
 - All input sanitized server-side (XSS, range checks)
 - Stateless architecture (easy to scale horizontally)
 - Express serves index.html as static file
 - CORS open for local development
